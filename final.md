# 대회 수행을 위한 4인 체제 임무 분담 및 운영 계획

## 1. 목적

본 문서는 대회 현장에서 로봇 자율주행 데모를 안정적으로 수행하기 위해  
4인 체제 기준으로 임무를 분담하고, 연습·예선·결승 과정에서 필요한 작업 절차를 정리하기 위한 문서이다.

현재 세부 담당자는 확정되지 않았으나, 대회 수행 과정에서 필요한 역할을 기준으로  
각 인원이 담당해야 할 업무 범위를 사전에 정의한다.

---

## 2. 대회 준비 핵심 목표

대회 준비의 핵심 목표는 다음과 같다.

1. A 맵과 B 맵 모두에서 자율주행 데모가 가능하도록 준비한다.
2. 대회장 환경에서 빠르게 맵을 만들고 포인트를 다시 정의할 수 있도록 한다.
3. 라이다, 젯슨 오린, 유전원 허브, 컨트롤러 등 하드웨어 상태를 안정적으로 유지한다.
4. 플래너 담당자와 협업하여 반복 데모를 수행하고 성공률을 높인다.
5. 실패 상황 발생 시 즉시 원인을 파악하고 복구할 수 있는 운영 체계를 만든다.

---

## 3. 4인 체제 역할 구성

아직 구체적인 담당자는 확정되지 않았으므로, 임시로 다음과 같이 역할을 구분한다.

| 역할 | 임시 명칭 | 주요 담당 업무 |
|---|---|---|
| 총괄 / 판단 담당 | 담당자 A | 전체 일정 관리, 최종 판단, 대회 수행 순서 결정 |
| 하드웨어 / 라이다 담당 | 담당자 B | 라이다, 전원, 유전원 허브, 로봇 구조물, 컨트롤러 관리 |
| 맵 / 포인트 담당 | 담당자 C | 맵 생성, 포인트 채집, A/B 맵별 좌표 관리 |
| 소프트웨어 / 플래너 협업 담당 | 담당자 D | goal generator, launch, planner 연동, 로그 확인 |

---

## 4. 역할별 상세 임무

## 4.1 담당자 A — 총괄 / 대회 운영 담당

### 주요 임무

- 전체 대회 일정 확인
- 예선 및 결승 수행 순서 관리
- 팀원별 작업 상황 확인
- 대회장 현장 판단
- 기능 추가 여부 최종 결정
- 데모 성공/실패 기준 관리
- 대회 수행 전 최종 체크리스트 확인

### 세부 작업

| 작업 | 내용 |
|---|---|
| 일정 확인 | 비기너 데이, 예선, 결승 일정 확인 |
| 역할 조율 | 각 담당자의 작업 우선순위 조정 |
| 최종 판단 | 대회 당일 코드 변경 여부 결정 |
| 기록 관리 | 데모 성공률, 실패 원인, 수정 내역 확인 |
| 대회 진행 | 실제 수행 전 준비 상태 최종 점검 |

### 담당자 A가 확인해야 할 핵심 질문

- 지금 대회 수행 가능한 안정 버전인가?
- 새 기능을 넣는 것이 필요한가, 위험한가?
- A 맵과 B 맵 중 어느 쪽 리스크가 더 큰가?
- 실패했을 때 바로 복구 가능한가?
- 대회장 환경에서 재현 가능한 상태인가?

---

## 4.2 담당자 B — 하드웨어 / 라이다 담당

### 주요 임무

- 라이다 설치 및 상태 확인
- 유전원 허브 전원 확인
- 젯슨 오린 연결 상태 확인
- 로봇 전체 전원 관리
- 라이다 구조물 고정
- 케이블 접촉 불량 확인
- 컨트롤러, 키보드, 마우스 준비

### 세부 작업

| 작업 | 내용 |
|---|---|
| 라이다 장착 확인 | 라이다가 수평으로 장착되어 있는지 확인 |
| 라이다 인식 확인 | `/scan` 토픽이 정상 출력되는지 확인 |
| TF 확인 | `base_link → laser` 변환이 정상인지 확인 |
| 전원 확인 | 로봇 배터리, 허브 전원, 젯슨 전원 확인 |
| 케이블 정리 | 라이다, 허브, 컨트롤러 케이블 고정 |
| 예비 장비 관리 | 예비 라이다, 케이블, 충전기 보관 |
